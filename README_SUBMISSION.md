# 业务数据智能分析平台 - 提交说明

## 项目简介

本项目是一个面向业务数据分析场景的 Web 原型系统，核心功能包括数据体检、清洗方案选择、图表分析、AI 追问结论和报告预览导出模拟。

## 推荐运行方式

请先安装 Node.js，然后在项目根目录运行：

```bash
npm install
npm run dev
```

启动成功后，在浏览器打开：

```text
http://127.0.0.1:5173/
```

## 静态构建版本

如果只需要查看构建后的网页，可以使用 `dist-static` 文件夹。建议通过静态服务器打开，例如：

```bash
npm install
npm run preview
```

然后访问终端中显示的本地地址。

## 文件夹说明

- `source-code/`：项目源码，不包含 `node_modules`。
- `dist-static/`：已经构建好的静态网页文件。
- `poster/`：项目展示海报，包含 9:16 PNG 海报和可编辑 HTML 源文件。
- `verification/`：测试和浏览器验证截图。

## 验证记录

已执行并通过：

- `npm test`
- `npm run build`
- `npm run lint`

说明：构建时 Vite 会提示 ECharts 相关 chunk 大于 500 kB，这是图表库体积带来的提示，不影响网页运行。

## 注意事项

`http://127.0.0.1:5173/` 是本机地址，不能直接作为给他人访问的公网网址。如果老师需要直接打开网页，需要部署到公网平台后再提交公网链接。
